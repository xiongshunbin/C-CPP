# C++ 学习
  本仓库存放的是我学习C/C++过程中所阅读的编程书籍以及所做过的项目的源码。

## 阅读的书籍
  1. 《C++ Primer Plus》
  2. 《C++ Primer》
  3. 《Effective C++》
  4. 《TCP/IP网络编程》
  5. 《C++并发编程实战》

## 做过的项目
  1. IB_System -- 智慧公交管理系统，该项目使用Qt5.15.2+MSVC构建，主要涉及的技术有: Qt 自定义控件, Qt 结合高德地图 API 进行实时地图显示，Qt 表格委托代理, MySQL 数据库编程，C++ 实现数据库连接池。
  2. TDGame -- 村庄保卫战，该项目使用SDL2游戏引擎，代码行数约4000+行，包含的核心技术有: 自定义Timer计时器类的实现，自定义Animation动画类的实现，CRTP单例模板类实现批量单例模式，使用cJson库进行json文件的解析，对csv文件格式的解析。通过本项目的锻炼，增强了我对于项目开发的整体观的把握的深刻理解和意识。
